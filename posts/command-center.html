Technical Architecture

Unlike standard web-based terminals, our Command Hub utilizes the Tauri IPC (Inter-Process Communication) bridge to execute system-level directives without the overhead of a dedicated shell process.

  • Direct Directive Execution: Commands like audit and manifest trigger native Rust functions in the execute_cli_directive module.
  • Session Persistence: Command history is logged to the Neural Ledger, allowing for temporal analysis of strategic shifts.
  • Stress-State Visuals: The terminal background and accent colors dynamically shift based on the OS "Stress Color," providing immediate visual feedback during critical operations.

Founder Access

The Command Hub can be summoned instantaneously via Ctrl + ` (Backtick) or Alt + T. This gesture-based invocation ensures a friction-less transition from 3D visualization to low-level strategic command.
